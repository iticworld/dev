# NOVEMBERIZING'S CONSOLE HEX VIEWER

__prerequisites__

 - [novemberizing c++](https://github.com/novemberizing/cplusplus)

```
git clone https://github.com/novemberizing/cplusplus.git
mkdir cplusplus/build
cd cplusplus/build
cmake -G "CodeBlocks - Unix Makefiles" ../
sudo cmake --build ./ --target install -- -j 2
```

__install__

```
wget https://raw.githubusercontent.com/iticworld/programming/master/util/hex/main.cpp
g++ -std=c++17 main.cpp -o hex -L/usr/local/lib -lnovemberizingc++
./hex main.cpp
```

__command__

```
hex "path"
```
